# NFT（非同质化代币）基础

**定位**：面向 Web3 初学者的 NFT 概念入门笔记，涵盖 NFT 是什么、同质化与非同质化的区别、版权问题、市场机制与风险。

---

## TL;DR

**NFT（Non-Fungible Token，非同质化代币）** 是区块链上的一种独一无二的数字凭证，用来证明某样东西（图片、音乐、游戏道具等）的**链上所有权归属**。与比特币或以太币不同，每个 NFT 不可互换——就像两张钞票可以互换，但两幅画的所有权凭证不可互换。

---

## 关键概念（中英对照）

- **NFT（Non-Fungible Token，非同质化代币）**：每个 Token ID 唯一、不可互换的链上凭证
- **同质化（Fungible）**：可以 1:1 互换，如 ETH、USDC；你的 1 ETH 和我的 1 ETH 没有区别
- **非同质化（Non-Fungible）**：每枚代币独一无二，有唯一 ID，不可替代
- **ERC-721**：以太坊上最主流的 NFT 标准，每个 Token ID 对应唯一 NFT
- **ERC-1155**：多代币标准，可在同一合约里同时管理同质化和非同质化代币（适合游戏道具）
- **元数据（Metadata）**：NFT 关联的描述信息，通常包含图片 URL、名称、属性等，大多存在链下（IPFS 或中心化服务器）
- **IPFS（星际文件系统）**：去中心化文件存储网络，常用于存储 NFT 图片和元数据
- **铸造（Mint）**：创建一个 NFT，将其写入区块链的过程
- **版税（Royalty）**：NFT 每次二级市场成交时，一定比例手续费自动转给原创者
- **地板价（Floor Price）**：某个 NFT 集合里当前最低的挂牌售价

## 买 NFT 等于买版权吗？

**不等于。** 这是最常见的误区：

| 你买到了什么 | 你没买到什么 |
|-------------|-------------|
| 该 NFT 的链上所有权凭证（Token ID） | 图片/音乐/视频的版权（Copyright） |
| 链上转让的权利 | 商业使用、复制、发布的权利 |
| 区块链记录的"拥有"记录 | 法律意义上的知识产权 |

版权归属需要另外看项目的**许可协议（License）**：
- 大多数 NFT 项目的版权归创作者，购买者只有持有和转售权
- 部分项目授予持有者一定的商业使用权（如 Bored Ape 允许持有者用自己的猿头开商标）
- 没有书面协议 = 版权不在你手上

**另一个核心问题：NFT ≠ 文件本身**
你买的 NFT 只是链上指向某个 URL 的凭证；图片本身通常存在链下（IPFS 或普通服务器）。如果服务器关闭，图片可能 404，但链上的 Token ID 依然存在——你拥有指向一个坏链接的凭证。

## NFT 的主要应用场景

1. **数字艺术（Digital Art）**：艺术家可以证明作品的"原作"所有权，并通过二级市场版税持续获益
2. **游戏道具（Gaming Items）**：NFT 道具跨游戏携带、真正属于玩家，而非游戏公司
3. **门票/会员资格（Ticket / Membership）**：用 NFT 做活动门票或俱乐部会员证
4. **数字收藏品（Collectibles）**：如 NBA Top Shot 球星卡、CryptoPunks
5. **音乐/版权分割**：音乐 NFT 让粉丝投资创作者，分享未来版税收益

## NFT 市场运作方式

- **一级市场（Primary Market）**：项目方直接向公众铸造/出售，通常通过 Mint（铸造）活动
- **二级市场（Secondary Market）**：买家在 OpenSea、Blur 等平台相互转售，版税按比例分给创作者
- **挂单（List）**：把你的 NFT 标价出售
- **出价（Offer/Bid）**：买家对某 NFT 或整个集合出价
- **扫底（Sweep the Floor）**：批量购买地板价 NFT，通常用于抬价

---

## 核心要点

1. NFT 证明的是链上所有权，不是版权或文件本身
2. NFT 的价值主要来自共识和稀缺性，没有内在保值属性
3. 元数据通常存在链下，项目方跑路可能导致图片丢失
4. 版税不是强制的，市场平台可以绕过；持有 NFT 不一定持续有版税收入
5. 智能合约决定规则，读懂合约比听项目方说更可靠

---

## 常见误区与风险

- **误区**：买了 NFT 就拥有了图片版权，可以随意商用
  **正确**：版权需要看许可协议，大多数 NFT 不附带版权转让

- **误区**：NFT 作为投资品一定会升值
  **正确**：大多数 NFT 集合的地板价最终趋近于零，流动性极差

- **风险**：**Rug Pull（跑路）**——项目方卖完 NFT 后关闭社区消失，图片服务器下线，社区炒作崩溃

- **风险**：**钓鱼（Phishing）**——假冒知名项目的 Mint 页面，签名后授权转走所有 NFT 和代币

- **风险**：**空投骗局**——收到不明来源 NFT，查看它的合约地址后中招
